# lcdc_cursor

## Overview
The lcdc_cursor example shows how to use LCD hardware cursor.

In this example, a 32x32 cursor is shown. The cursor's position is changed at
the end of every frame.

The background is:
+------------------------------------------------+
+                                                +
+                    Red                         +
+                                                +
+            +--------------------+              +
+            +                    +              +
+            +      Blank         +              +
+            +                    +              +
+            +--------------------+              +
+                                                +
+                                                +
+                                                +
+------------------------------------------------+
