# pca9420

## Overview
The pca9420 driver example demonstrates the usage of pca9420 SDK component driver.
The example shows the usage of PF1550 API to:
1. Dumping Mode Settings;
2. Switch Mode;
3. Dump PCA9420 register content;
4. Feed watchdog;
