# rom_dev_cdc_bm

## Overview

The example shows how to use USBD ROM stack to create a virtual comm port. When
the USB on J3 is connected to a PC, the host would recognize the USB connection
as a new serial port.

## Supported Boards
- LPCXpresso51U68
- LPCXpresso54628
- LPCXpresso54S018
- LPCXpresso54S018M
