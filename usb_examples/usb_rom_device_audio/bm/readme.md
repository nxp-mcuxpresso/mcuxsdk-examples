# rom_dev_audio_bm

## Overview

The example shows how to use USBD ROM stack to create and USB AUDIO class
device. This device supports 2 channel audio in (MIC/LINE_IN) and 2 channel
audio out (SPEAKERS/HEADPHONE).

## Supported Boards
- LPCXpresso54S018
- LPCXpresso54S018M
- LPCXpresso54628
- LPCXpresso51U68
